Nova is a menu bar widget for OS X that displays crypto-currency tickers. 

Simply select the coins you wish to follow and watch them change in your menu bar!

Data for the app is provided by [CryptoCompare](https://www.cryptocompare.com/api/) and [CoinMarketCap](https://coinmarketcap.com/api/), thanks!
